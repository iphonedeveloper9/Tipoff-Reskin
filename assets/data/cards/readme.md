## Excel to JSON
`Add-ons > Export Sheet Data > Open Sidebar`

#### Format
###### Export Format
> JSON
###### Export Folder
> Default
###### Export Sheets
> Current sheet only

#### Advanced JSON
###### Export contents as array
> (CHECKED)
###### Empty value format
> `Empty String ("")`
###### Null value format
> `String ("null")`
